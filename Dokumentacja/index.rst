Welcome to Aero - Dziennik Rowerzysty's documentation!
======================================================

Dane techniczne:
Program Aero - Dziennik Rowerzysty korzysta z biblioteki Tkinter oraz bazy danych sqlite3.
Wszystkie grafiki pobierane sa z folderu glownego w formacie ".ppm". Program do poprawnego
dzialania wymaga systemu Apple OS X z pakietem czcionek Helvetica i programem Python w wersji 2.7.

Instrukcja uzytkowania
Program uruchamia sie poprzez plik "main.py". Poprawne uruchomienie programu ukazuje jego logo.
"Aero" sklada sie z 4 zakladek w tym domyslna "Aktualnosci". Przy pierwszym uruchomieniu programu nalezy udac sie do zakladki "Ustawienia"
i wcisnac przycisk "Zmien dane". Wyskoczy okienko ktore umozliwi zmiane danych na swoje. Jesli to konieczne nalezy rowniez
wyzerowac dane treningow i celow przyciskiem "Wyzeruj program". Po takiej konfiguracji program jest juz gotowy do uzycia.

- Dodaj trening
Aby dodac trening nalezy udac sie do zakladki "Moj dziennik" i nacisnac przycisk "Dodaj trening". Wyskoczy okno umozliwiajace
dodanie nowego treningu. Po dodaniu danych nalezy zatwierdzic przyciskiem "Zapisz" lub wyjsc nie zapisujac danych przyciskiem "Anuluj".

- Uzyskanie celu
Po uzyskaniu pulapu wybranego celu program sam zresetuje dane pod nowy cel i doda wartosc 1 do uzyskanych celow.
